TCP FILE TRANSFER - TESTING GUIDE

TESTING STEPS:
1. Start server in one terminal
2. Start client in another terminal
3. Test each menu option:
   - List files
   - Download a file
   - Upload a file
   - Get file info
   - Delete a file
   - Rename a file
   - Create directory
   - List directories

EXPECTED RESULTS:
- Server shows logs for each operation
- Client shows progress and success messages
- Files transfer correctly
- Error messages for invalid operations
